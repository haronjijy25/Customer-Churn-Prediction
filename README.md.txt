# Customer Churn Prediction & Revenue Risk Engine

This project predicts customer churn and calculates revenue at risk using Machine Learning.

## Features
- Customer churn prediction
- Revenue risk calculation
- Random Forest model
- Data analysis & visualization
- SQL + Python integration

## How to run
1. Install dependencies:
   pip install -r requirements.txt

2. Setup database using SQL scripts

3. Run:
   python churn_full_project.py

## Output
- Churn probability for each customer
- Revenue at risk
- Ranked high-risk customers
